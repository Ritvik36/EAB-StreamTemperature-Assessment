# EAB-StreamTemperature-Assessment
Hello and welcome to EAB-StreamTemperature-Assessment!
This page contains scripts and protocols for assessing the impact of Emerald Ash Borers on stream temperatures in subwatersheds of the Willamette Valley, using the Marys River Watershed as a case study. This includes the remote identification of Oregon ash trees using multispectral satellite data and a Random Forest model.

Scripts are organized into the following sections:
-	Multispectral Supervised Classification 
o	Solar Angles Function
o	Preprocessing Preparations
o	Sentinel Topographic Corrections
o	Random Forest Classifier
o	Validation Point Sampling

Required user inputs will be located towards the top of each script in a designated segment. Refer to code comments for information on inputs and lines of code. Refer to the following graduate thesis for detailed information on the overall methodology, process, and procedures: https://ir.library.oregonstate.edu/concern/graduate_thesis_or_dissertations/bv73c8453?locale=en
For Multispectral Supervised Classification, refer to the provided Supplementary Protocol Instructions for further information on the scripts and inputs
